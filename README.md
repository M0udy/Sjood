# Sjood

A prayer focus application built with React Native, Expo, and Anthropic API.

## Features

- Prayer tracking and reminders
- Islamic calendar integration
- AI-powered prayer guidance
- Multi-language support
- Offline functionality

## Tech Stack

- **Mobile**: React Native + Expo
- **Backend**: Next.js API Routes (optional)
- **AI**: Anthropic API
- **Database**: Supabase (optional)
- **Authentication**: Supabase Auth (optional)

## Getting Started

### Prerequisites

- Node.js 16+
- Expo CLI (`npm install -g expo-cli`)
- Yarn or npm

### Installation

```bash
npm install
# or
yarn install
```

### Environment Setup

```bash
cp .env.example .env.local
# Edit .env.local with your configuration
```

### Running Locally

```bash
# Start Expo development server
expo start

# On iOS simulator: press 'i'
# On Android emulator: press 'a'
# For web: press 'w'
```
